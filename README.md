# BLINKY1
Blink.Pause.Repeat
First hardware project ✅ 
I Created my first blinky project with use of breadboard and it was super fun organised by Build Guild Bhopal.
 This is a simple breadboard circuit witha few LEDs.
 In this hands on session , participants will create a blinking LED circuit using a breadboard and basic components. This project introduces fundamental concepts of electronics like timing, current flow, and circuit design.
BOM:LEDs · Resistors · Jumper Wires · Breadboard · 9V Battery
 I used:
 Resistor
 Jumper wire
 Breadboard
 LEDs
 Power supply( battery)
 Huge thanks to Build Guild Bhopal for organizing such a beginner-friendly electronics session!
